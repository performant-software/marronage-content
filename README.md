# Marronage Content

Content repository for the Marronage Mapping Project website, built with Core Data Places.

## Setup

This repo provides configuration, branding, and content for a Core Data Places deployment. See the [CDP documentation](https://github.com/performant-software/core-data-places) for deployment instructions.

## Structure

- `content/settings/config.json` — Site configuration (search, maps, API)
- `content/branding/branding.json` — Visual identity
- `content/i18n/en.json` — UI translations
- `content/pages/` — Static pages (MDX)
- `content/paths/` — Curated narrative tours
- `content/posts/` — Blog/news posts
- `content/users/` — User configuration
